## 🎨 視覺設計系統

### 色彩配置
```css
主色調: #667eea (藍紫色)
次要色: #764ba2 (深紫色)
強調色: #f093fb (粉紅色)
漸層背景: linear-gradient(135deg, #667eea, #764ba2, #f093fb)
```

### 字體系統
```css
中文字體: 'Noto Sans TC' (Google Fonts)
英文字體: 'JetBrains Mono' (程式碼展示)
字體大小: 14px-56px (響應式調整)
```

### 動畫系統
- 載入動畫: 2秒進入效果
- 滾動動畫: IntersectionObserver觸發
- 懸停動畫: 0.3秒過渡效果
- 粒子動畫: Canvas動態背景
